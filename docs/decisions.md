# Engineering Decisions

آخر تحديث: 2026-07-22

---

# القرار 001

العنوان:
فصل Laravel عن Huawei.

القرار:

Laravel لا يتصل بالمودم مباشرة.

يتم الاتصال عن طريق Node.js Agent فقط.

السبب:

- زيادة الأمان.
- سهولة التطوير.
- إمكانية تشغيل عدة Agents.

الحالة:

✅ معتمد.

---

الحالة:

✅ معتمد.

# القرار 003

✅ معتمد.

# قواعد ثابتة
